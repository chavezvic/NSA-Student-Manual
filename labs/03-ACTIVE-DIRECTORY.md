# Lab 03: Active Directory Domain Services

## Objective

Create a small Active Directory environment with one domain controller and one domain-joined workstation.

## Required work

- Create an AD design.
- Install AD DS and DNS.
- Promote the server.
- Create OUs, users and security groups.
- Join a workstation.
- Apply one test GPO.
- Validate DNS, authentication and policy application.
- Run `dcdiag`.
