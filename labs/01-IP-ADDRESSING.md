# Lab 01: IP Addressing and Validation

## Objective

Configure two systems in the same IPv4 subnet and prove end-to-end connectivity.

## Deliverables

- Addressing table
- Network diagram
- Configuration steps
- Ping evidence
- ARP or neighbor table evidence
- Reflection

## Success criteria

- Both systems have correct unique IP addresses.
- Both systems use the same subnet mask.
- Both systems communicate.
- The student explains why no router is required for same-subnet communication.
