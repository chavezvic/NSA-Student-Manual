# Student Roadmap

## Phase 1: Foundations

Learn:

- Hardware components, firmware, BIOS and UEFI
- Windows and Linux installation
- Filesystems, permissions and processes
- IPv4 addressing and subnetting
- Git and Markdown basics

Evidence:

- Build sheet
- IP addressing table
- OS installation record
- First Git repository

## Phase 2: Network services

Learn:

- Ethernet and switching
- VLANs, trunks and inter-VLAN routing
- Static and dynamic routing concepts
- DNS, DHCP and NAT
- Packet capture and protocol analysis

Evidence:

- Logical network diagram
- Switch configuration
- DHCP scope
- DNS forward and reverse lookup tests
- Wireshark capture with explanation

## Phase 3: Windows infrastructure

Learn:

- Active Directory forests, domains, sites and OUs
- Users, groups and computers
- Group Policy
- DNS integration
- DHCP authorization
- File services and permissions
- PowerShell administration

Evidence:

- AD design
- OU and group strategy
- GPO inventory
- PowerShell script
- Backup and restore test

## Phase 4: Linux infrastructure

Learn:

- Package management
- Users, groups and sudo
- systemd services
- SSH
- Storage and filesystems
- Web and file services
- Bash automation

Evidence:

- Hardened Linux server
- Service runbook
- Bash script
- Log analysis report

## Phase 5: Virtualization and cloud

Learn:

- Hypervisors
- Virtual networks
- VM sizing
- Templates, clones and snapshots
- High availability concepts
- Backup and recovery
- Containers and cloud fundamentals

Evidence:

- Virtual environment diagram
- VM inventory
- Backup schedule
- Restore test
- Capacity plan

## Phase 6: Security and operations

Learn:

- Asset inventory
- Least privilege
- Patch and vulnerability management
- Logging and monitoring
- Incident handling
- Change management
- Business continuity

Evidence:

- Risk register
- Hardening checklist
- Incident report
- Change request
- Recovery procedure

## Phase 7: Professional portfolio

Complete:

- Architecture diagram
- Executive summary
- Technical implementation guide
- Troubleshooting case study
- Automation script
- Security assessment
- Presentation
