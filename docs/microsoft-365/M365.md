# Microsoft 365, Identity and Messaging

## Core areas

- Tenant administration
- Microsoft Entra ID
- Users, groups and licensing
- Authentication and multifactor authentication
- Conditional Access
- Exchange Online
- Mail flow
- Domains and DNS records
- Defender
- Compliance and auditing
- Hybrid identity and messaging concepts

## Administrative habits

- Use a separate privileged account.
- Apply least privilege.
- Record license changes.
- Verify DNS before troubleshooting mail flow.
- Check service health.
- Use message trace and audit logs.
- Document connectors, transport rules and accepted domains.
- Protect emergency access accounts.
- Never place secrets in scripts or repositories.

## Exchange checks

```powershell
Get-AcceptedDomain
Get-ReceiveConnector
Get-SendConnector
Get-Mailbox
Get-Queue
Test-Mailflow
```

Command availability depends on the Exchange environment and connected module.
