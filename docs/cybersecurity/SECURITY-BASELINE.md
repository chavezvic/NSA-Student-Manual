# Security Baseline

## Govern

- Define owners.
- Maintain policies.
- Record risks.
- Approve exceptions.
- Review third parties.

## Identify

- Inventory hardware, software, identities and data.
- Classify assets.
- Identify dependencies.
- Assess vulnerabilities and risk.

## Protect

- Use multifactor authentication.
- Apply least privilege.
- Patch systems.
- Segment networks.
- Encrypt sensitive data.
- Back up critical systems.
- Train users.

## Detect

- Centralize logs.
- Monitor authentication.
- Alert on suspicious activity.
- Establish normal baselines.
- Test detection rules.

## Respond

- Maintain an incident response plan.
- Preserve evidence.
- Contain the incident.
- Communicate through approved channels.
- Record decisions.

## Recover

- Restore from tested backups.
- Validate integrity.
- Communicate status.
- complete a lessons-learned review.
- Update controls and documentation.
